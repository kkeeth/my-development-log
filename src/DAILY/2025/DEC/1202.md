# 1202

## 濁流

### 今日は毒を吐く日なので閲覧注意

- 今日はかなりネガティブな愚痴の日
- 逆にここで毒をしっかり吐ききれればスッキリすると思うのでやってみる
- DEVELOPMENT の方にも書いたが，自分が半年後，1 年後に，今の部長や VPoE のレベルに上り詰める未来は見えない
  - まぁそれは自分の成長への挑戦や，試行錯誤など，圧倒的にアクションの量が少ないのだが
  - 行動を変えないで結果も成長もないわけで
  - 小さくても良いので少しでも，ダサくても，泥臭くても何か行動をし続けるしかない
  - 読書・インプットも同じ
  - 今の自分の中に新しい情報や知識が入ってこないなら，自分の中から何かが変わる可能性は全くのゼロだ
    - 組織も人も，新しい風は大事
    - 自分だけで成長することはないことは自然界が証明しており，それは単なる細胞分裂
    - なにか新しい刺激があるからこそ，今までにない視点と考えが浮かぶもの
  - このくだりは単なる自分のだらけをグチグチ書いているだけ
- とはいえ，自分の考えと向こうの考えの違いがあったのだが，たたきを作った自分側が悪いみたいな流れはハッキリ行って受け入れがたい
  - 自分から場を用意して，自分から時間を作って，自分からドキュメントを読ましたので，リソースをかなり奪っているといえば奪っている
  - それは自覚しているが，納得がいかねぇ
  - こちらとしては申し送りをしなくても良かったと思う
  - 受け入れ側から聞きたいことを本人またはマネージャーの自分に聞いてこいや，という気持ちだった
    - 先に何が聞きたい？という探りを入れておけばよかったという反省はあれど，これじゃあ全然足りねぇ，という上から目線は「はっ？」となったわ
  - まぁでもこれは今まで自分が他者の感情や気持ちを考えること，自分の頭の中野言語化をサボってきたせいでもあるので，必要な痛みでもある
    - この違和感，ストレスを乗り越えた先に成長があるので，その機会をいただいたと受け入れるところが第一歩
    - まだまだ自分は子供なんだと理解すること
